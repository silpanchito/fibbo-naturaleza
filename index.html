¿Qué problema real resolvemos?

En la naturaleza, las plantas no crecen de forma aleatoria. Sus hojas, pétalos y ramas siguen un patrón matemático preciso conocido como la serie de Fibonacci, que les permite maximizar la captación de luz solar y optimizar el espacio disponible.

Este proyecto analiza cómo se manifiesta dicha serie en el crecimiento de niveles de una planta: cada nuevo nivel de ramas sigue exactamente la progresión 1, 1, 2, 3, 5, 8, 13…

Comprender este patrón permite a botánicos, diseñadores y arquitectos recrear estructuras naturales eficientes en sus proyectos.

¿Cómo funciona el algoritmo?

01

La Serie de Fibonacci

Empieza con los valores 0 y 1. Cada número siguiente es la suma de los dos anteriores:
0 → 1 → 1 → 2 → 3 → 5 → 8 → 13 → 21…

02

Aplicación en Plantas

Cada nivel de crecimiento de una planta representa un término de la serie. El número de ramas en cada nivel sigue exactamente esta progresión matemática.

03

El Algoritmo en JavaScript

Usamos dos variables simples (a y b) y un ciclo for. Sin arreglos: solo sumas sucesivas. El resultado se muestra en pantalla usando getElementById().

0 + 1 → 1 → 2 → 3 → 5 → 8 → ...

Conclusión

La serie de Fibonacci no es solo una curiosidad matemática: es el lenguaje con el que la naturaleza optimiza el espacio. Las plantas que siguen este patrón logran que sus hojas no se tapen entre sí, maximizando la exposición solar de cada una.

Este proyecto demuestra que con apenas dos variables y un ciclo repetitivo, podemos modelar comportamientos observados en millones de especies vegetales. La matemática no describe la naturaleza: la naturaleza habla matemática.

Aplicaciones prácticas de este conocimiento incluyen el diseño arquitectónico biofílico, la optimización de paneles solares y el modelado de cultivos agrícolas.
